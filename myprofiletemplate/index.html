Final Project

Arguepedia

The task was to create a website that is part questionnaire and part digital encyclopedia. The point is that you can navigate the website and choose a subject. The subject would be laid out as an argument with a particular semantic structure that minimises the abstractness of ordinary language and therefore minimises the broad range of answers that can be given to questions those questions. The point was that using the structure would effectively allow us to retrieve precisie information and the answers would be categorised in such a way that other a similar structured questions would be applied to them and therefore in theory would have the ability to endlessly connect the answers to more questions to more answers. The website prototype was built using object oriented programming, model view controller (MVC) structure with the asp.net framework, C#, sql, html and css.
